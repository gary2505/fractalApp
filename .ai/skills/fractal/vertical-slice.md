# Skill: fractal vertical slice

Use for new feature work.

Rule:
Build one real vertical slice, not many skeletons.

Slice must include:
- contract
- state/input
- UI
- messages
- small check
- rollback note

Keep small:
- one feature
- one flow
- one visible result

Forbidden:
- global redesign
- unrelated cleanup
- future placeholders without use
- many empty folders/apps
