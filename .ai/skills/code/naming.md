# Naming

Keep names short.

## Rules

```txt
folder: lowercase kebab or one word
file: short kebab-case
block: block-xxx
no repeated words
no long history names
```

## Bad / good

```txt
bad: block-system-foundation-from-filesup
good: block-system

bad: src/settings/setting-session/session-store.ts
good: src/settings/store.ts

bad: src/system/input/click-controller.ts
good: src/system/input/click.ts

bad: BLOCK-MANIFEST.js
good: BLOCK-MANIFEST.json
```

If name explains history, shorten it.
